프로젝트 개요
=============

ReviewEat은 기존 지도앱, 블로그, 메모장 등이 가진 한계를 보완하여 사용자가 개인화된 맛집 탐색과 프라이빗 리뷰 작성을 효율적으로 할 수 있도록 지원합니다.

Google Maps API를 활용한 정확한 위치 정보와 Flutter를 이용한 크로스 플랫폼 지원을 특징으로 합니다.

주요 특징
---------

* 지도 기반 실시간 음식점 검색 및 탐색
* 개인화된 리뷰 작성 및 관리
* 달력형 리뷰 일기장
* 인스타그램 스타일 스토리 뷰
* 크로스 플랫폼 지원 (iOS, Android)
